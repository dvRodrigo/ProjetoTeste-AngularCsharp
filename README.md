
## Tecnologias Usadas

- **Frontend**: Angular 17
- **Backend**: WebAPI C# (minimal) => Pacote Bugos para geração de dados fake.

## Instalação

1. Baixe o arquivo ZIP

2. Navegue até o primeiro diretório do projeto:

 ```bash
   dotnet run
    ```

3. Navegue até a pasta frontend-pessoas:

    ```bash
    npm install
    ```

4. Execute o projeto:

    ```bash
    ng serve
    ```

5. Abra o navegador e acesse:

    ```
    http://localhost:4200
    ```
